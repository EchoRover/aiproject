# Datasets Folder

Store all datasets here. Organize by project/domain if using multiple datasets.

## Structure:
- Create subfolders for each dataset
- Include data sources and descriptions
- Keep raw and processed data separate if needed

Example:
```
datasets/
├── dataset1_name/
│   ├── raw/
│   ├── processed/
│   └── README.md (description and source)
```
